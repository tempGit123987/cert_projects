C++

Ah, good ol C++, nothing beats that (ironically that reference is probably dated to people that don't have fond memories of C++ but oh well).

C++ is the starting point for a lot of programmers, and for good reason, it has just the right amount of readability, but also technical strength, here let me show you.

Hello world

The classic hello world, basically the first thing anyone ever learns to write in any language. You might not know it now, but you'll feel real nosalgic for this one day, trust me. Anyway to write it in C++ you would simply type this:

cout << "Hello World"

That's it! ...well sort of. There is more to it than that, but one step at a time. "cout" stands for "console out" and it means somethign being output to the console, pretty simple huh? Don't worry, this is literally only the tip of the iceburg.
